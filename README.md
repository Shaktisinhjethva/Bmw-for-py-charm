# 📖 Overview

Biomedical waste is one of the most critical challenges in healthcare. Improper disposal can spread infections, pollute the environment, and violate government regulations.

This project provides a **digital waste management platform** where hospitals can record waste generation, request waste collection, monitor pickup vehicles, and maintain complete records while certified waste collectors safely transport waste to treatment facilities.

The system increases transparency, improves operational efficiency, and supports environmentally responsible waste disposal.

# 💡 Problem Statement

Traditional biomedical waste management relies heavily on manual documentation, delayed communication, and limited transparency.

Common issues include:

- Poor waste tracking
- Delayed pickups
- Data inconsistency
- Manual paperwork
- No real-time monitoring
- Compliance challenges

This project solves these problems through complete digital automation.

# 🛠 Installation

Clone Repository

```bash
git clone https://github.com/Shaktisinhjethva/Bmw-for-py-charm.git
```

Go to project

```bash
cd Bmw-for-py-charm
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Environment

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Server

```bash
python app.py
```

Open Browser

```
http://127.0.0.1:5000
```
